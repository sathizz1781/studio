# **App Name**: WeighBridge Biller

## Core Features:

- Billing Data Input: Input fields for: serialNumber, date and time, vehicleNumber, partyName, materialName, charges, firstWeight, secondWeight, netWeight.
- Fetch Serial Number: Fetch the latest serial number from an external API.
- WhatsApp Billing Notification: Send a WhatsApp message to the entered phone number with the billing details. This feature acts as a tool using logic based on provided data to produce user-specified behavior
- Data Presentation: Display all data clearly.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Light gray (#ECEFF1) for a clean, professional look.
- Accent color: Teal (#009688) to highlight important actions.
- Font: 'Inter', a sans-serif font for clear readability and a modern look, for both headlines and body.
- Use simple, consistent icons to represent data entry fields and actions.
- A clean, organized layout with clear sections for data input and display.
